Keep Dry Run on for the first run. Turn it off only after the terminal preview looks correct.

Dry Run - preview only No files are changed. The system only reads data, matches rows and prints planned changes.
Create Backup Reserved in the payload for future backup behavior. Current operation logic does not create backups yet.
Overwrite existing values When off, existing target/master cells are protected and skipped.
Trim values before matching Removes extra spaces from keys before comparing. Recommended for most Sheets work.
Case sensitive matching When on, “Hotel A” and “hotel a” are treated as different keys. Usually keep this off.
